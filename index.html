Conclusion

If we look at the individuals franchise game and the score(not the average score), we can see that there is a correlation between score and sale, even though it doesn't match 100%.
When the score rise most of the time the sale will also rise. And vice versa

So, inclusion sale number can be predicted with the score, as long as game has a franchise and is in a action genre.

With this franchise for the games like uncharted and Zelda we can see that there is correlation between sale and score.
With Uncharted that sale and score is behaving in similar manner. And with Zelda only the last few year show the best correlation.

So, inclusion sale number and score has a connection with each other.

It seems that with the franchise racing the score and sale doesn't really has a match with each other. When we take a look at the individuals game franchise the pattern of sale and score seems to be off.

So, with that be said we can assume that the racing genre with franchise doesn't have a good correlation. It can be that game is more of fan base.

In general the score and sale in game franchise in shooter does have a matching pattern. But if we look at game like call of duty it seems that this franchise is very sensitive too the score of game.

So, inclusion just like action and adventure the score does have an affect on the sale number. But some franchise seems to be more sensitive than other franchise to the score.

It seems that the only game franchise that the sale and score has the same pattern seems to be Tekken. So, with these information it shows that Tekken is the only franchise that is influenceable by the score.

In conclusion franchise in action and adventure genre has a better co-relation with sale and score. If you going to invest in game franchise base of game score, it is better to do so with these genre.
This doesn't mean that you should avoid other franchise, it is just that with other franchise your game should have a stable fans base.
